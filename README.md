# angular-node-ping-pong
An app that allows you to add ping pong matches and see a leaderboard for current players.

## The stack
- Node
- Express
- Mongo
- Angular
- Sass
- Gulp
- Bower

## Instal
```sh
$ git clone git@github.com:elmargasimov/angular-node-ping-pong.git
$ npm install
$ bower install
```

## Running the app
```sh
$ npm run app
$ gulp
```
